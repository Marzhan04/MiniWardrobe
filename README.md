README.md

Overview
MiniWardrobe is a website designed to simplify daily outfit selection for users. It generates  outfits based on a user's wardrobe and preferences, helping to save time and reduce decision fatigue.

The product solves the problem of daily outfit planning by providing users with pre-generated outfit suggestions from their existing wardrobe.

The target user group includes students, young professionals, and fashion enthusiasts who wish to streamline their daily wardrobe choices.

Tech Stack:

- Front end: HTML, CSS, JavaScript
- Back end: Flask (Python)
- Database: SQLite
- Other tools: Figma (for design), GitHub (for version control)

Project Structure:

- /src: Source code for the application, including front-end and back-end files.
- /tests: Test files for unit testing and integration testing.
- /docs: Documentation related to the project.

How to Run the Project:

System requirements:
- Python 3.x
- SQLite (or any relational database)
- Flask

Installation steps:

1. Clone the repository:
  git clone <repository_url>

2. Navigate into the project directory:
  cd MiniWardrobe

3. Install dependencies:
  pip install -r requirements.txt

Start command:
flask run

How to Run Tests

Test command:
pytest
