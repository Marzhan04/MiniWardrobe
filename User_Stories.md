User Stories

Story 1
As a user, I want to add a new wardrobe item, so I can keep track of my clothing.

Acceptance Criteria:
- User can add items with a name.
- Items are displayed in the wardrobe.

Story 2
As a user, I want to generate an outfit using my wardrobe, so I can quickly decide what to wear.

Acceptance Criteria:
- The system generates an outfit from the wardrobe.
- The generated outfit is displayed with item names.
